# NLP Emotion Detection Web Application

A simple web application for detecting emotions from user text using Natural Language Processing (NLP). The application provides an interactive interface where users can submit text and receive emotion analysis results in real time.

---

## Overview

This project demonstrates the integration of a web-based user interface with an emotion detection backend. Users enter a piece of text, and the application sends it to the server for analysis before displaying the detected emotions.

---

## Features

* Text-based Emotion Detection
* Interactive Web Interface
* Real-Time Analysis
* AJAX-Based Communication
* Bootstrap User Interface
* Fast Response without Page Reload

---

## Tech Stack

* HTML5
* CSS3
* JavaScript
* Bootstrap
* AJAX (XMLHttpRequest)
* Python (Backend)
* Natural Language Processing (NLP)

---

## How It Works

1. Enter a sentence or paragraph.
2. Click **Run Sentiment Analysis**.
3. The application sends the text to the backend.
4. The backend performs emotion detection.
5. The detected emotions are displayed instantly on the page.
